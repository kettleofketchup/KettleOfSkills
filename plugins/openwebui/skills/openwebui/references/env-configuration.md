# Environment Variables Reference

This reference has been split into separate files for comprehensive coverage (v0.8.5):

- **env-general.md** - General app config, directories, logging, AIOHTTP client, streaming, banners
- **env-auth.md** - OAuth, OIDC, Google/Microsoft SSO, security, cookies, API keys, password validation
- **env-rag.md** - RAG, embedding, content extraction, vector DB config, web search, document processing
- **env-connections.md** - Ollama, OpenAI, database, Redis, WebSocket, tool servers, vector databases (ChromaDB, PGVector, Qdrant, Milvus, etc.)
- **env-media.md** - Audio (STT/TTS), Whisper, ElevenLabs, Azure Speech, image generation (DALL-E, ComfyUI, A1111, Gemini)
- **env-features.md** - Tasks, code execution, code interpreter, autocomplete, pip install, performance profiles
